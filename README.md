# agentic-e2e-workflow

Agentic, AI-powered end-to-end quality guardian for web applications.

## Goals
- Execute critical user flows
- Detect UI, functional, and logical regressions
- Use LLM reasoning on top of browser automation
- Produce actionable, human-readable reports

## Tech Stack
- Node.js + TypeScript
- Playwright
- LLM Provider (OpenAI first)
